## 🎯 الخطوات الأساسية للاستخدام

### 📝 تسجيل الدخول كمدير
1. اذهب إلى http://localhost:5174/login
2. اختر "المدير"
3. أدخل البيانات:
   - البريد: `admin@copysearch.dz`
   - كلمة المرور: `admin123`
4. اضغط "تسجيل الدخول"

### 📊 لوحة تحكم المدير
من لوحة المدير يمكنك:
- **إضافة طالب جديد**: ادخل اسم الطالب والنقاط الأولية
- **إضافة محل تجاري**: ادخل البيانات الكاملة
- **إدارة النقاط**: عدّل نقاط أي مستخدم
- **حذف المستخدمين**: احذف الطلاب أو المحلات

### 👨‍🎓 تسجيل الدخول كطالب
1. اذهب إلى http://localhost:5174/login
2. اختر "الطالب"
3. أدخل رمز الدخول: `H21U3V1`
4. اضغط "تسجيل الدخول"

### 📚 لوحة الطالب
من لوحة الطالب يمكنك:
- **توليد بحث جديد**: ادخل عنوان البحث والبيانات
- **عرض السجل**: شاهد الأبحاث السابقة
- **تحميل الملفات**: حمل أبحاثك بصيغة Word
- **مولد المطلب**: وليد مطالب فردية

## 🔍 استكشاف الأخطاء

### إذا لم يعمل السيرفر

#### المشكلة: `EADDRINUSE`
```bash
# ابحث عن العملية على المنفذ 3001
netstat -ano | findstr :3001

# أقتل العملية (استبدل PID برقم العملية)
taskkill /PID <PID> /F
```

#### المشكلة: خطأ في الاتصال بـ MongoDB
```bash
# تأكد من أن MongoDB قيد التشغيل
mongosh --version

# شغل MongoDB
mongod
```

#### المشكلة: خطأ 401 في الكونسول
- تأكد من أنك سجلت دخولك بشكل صحيح
- افحص localStorage في المتصفح (F12 -> Application)
- جرّب تسجيل الخروج ثم تسجيل الدخول مجدداً

### افحص الأخطاء في الكونسول
1. افتح المتصفح (F12)
2. اذهب إلى تبويب Console
3. ابحث عن الأخطاء (باللون الأحمر)
4. اقرأ رسالة الخطأ لفهم المشكلة

## 🆘 الدعم والمساعدة

إذا واجهت أي مشاكل:

1. **افحص سجلات الأخطاء**:
   - اضغط F12 في المتصفح
   - انظر إلى تبويب Console
   - ابحث عن رسائل الخطأ

2. **اقرأ السجلات في Terminal**:
   - انظر إلى رسائل الخادم (يبدأ بـ [0])
   - ابحث عن الأخطاء والتحذيرات

3. **أعد تشغيل الخادم**:
   - أغلق Terminal
   - شغل `npm run dev` مجدداً

4. **مسح الـ Cache**:
   - حذف folder `node_modules`
   - حذف `package-lock.json`
   - قم بـ `npm install` مجدداً
